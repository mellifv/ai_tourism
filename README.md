# Astana AI Tourism

Welcome to the Astana AI Tourism project! This application is designed to provide users with AI-curated travel itineraries for exploring Astana, Kazakhstan. 

## Project Overview

The Astana AI Tourism application leverages modern web technologies to deliver a seamless and interactive user experience. It features a futuristic landing page, intelligent itinerary generation, and showcases popular experiences in Astana.

## Project Structure

- **public/index.html**: The main HTML file that serves as the entry point for the application.
- **src/components/astana_ai_landing_prototype.jsx**: Contains the `AstanaAI_LandingPrototype` React component, which previews the landing page.
- **src/pages/Home.jsx**: Exports the `Home` component, serving as the main page of the application.
- **src/styles/globals.css**: Global CSS styles, including Tailwind CSS configurations.
- **src/App.jsx**: The main `App` component that sets up routing and layout.
- **src/main.jsx**: Entry point for the React application, rendering the `App` component.
- **package.json**: Configuration file for npm, listing dependencies and scripts.
- **vite.config.js**: Configuration for Vite, specifying build and serve options.
- **tailwind.config.cjs**: Configuration for Tailwind CSS, defining themes and plugins.
- **postcss.config.cjs**: Configuration for PostCSS to process CSS files.
- **jsconfig.json**: Configuration for JavaScript projects, aiding module resolution.
- **.gitignore**: Specifies files and directories to be ignored by Git.

## Getting Started

To get started with the Astana AI Tourism project, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd astana-ai-tourism
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the application**:
   ```
   npm run dev
   ```

4. **Open your browser** and navigate to `http://localhost:3000` to view the application.

## Features

- AI-generated itineraries based on user preferences.
- Interactive and visually appealing landing page.
- Showcase of popular experiences in Astana.
- Responsive design for optimal viewing on various devices.
